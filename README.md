## Setup

### Prerequisites
- Python 3.10+
- Git

### 1. Clone the repo
```bash
git clone <repo-url>
cd cs539_piedpiper
```

### 2. Create and activate a virtual environment
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install Python dependencies
```bash
pip install -r requirements.txt
```

### 4. Build the Zig shared library
```bash
python -m ziglang build
```
This compiles `src/main.zig` and outputs:
- **Windows:** `zig-out/bin/piedpiper.dll`
- **macOS:** `zig-out/lib/libpiedpiper.dylib`
- **Linux:** `zig-out/lib/libpiedpiper.so`

### VSCode Setup (optional but recommended)

Install the **Zig Language Extension** in VSCode:
- Open Extensions (`Ctrl+Shift+X`)
- Search for `ziglang.vscode-zig` and install it

Then point the extension at the Zig binary bundled in your venv.
Find your venv's Zig path by running:

```bash
python -c "import ziglang; print(ziglang.__file__)"
```

This prints something like:

```bash
C:\Users\yourname\project\venv\Lib\site-packages\ziglang_init_.py
```

The `zig.exe` is in that same folder:
```
C:\Users\yourname\project\venv\Lib\site-packages\zig.exe
```
In VSCode, go to **Settings → search `zig.path`** and paste the full path to `zig.exe` in the **Zig: Path** field.

When prompted by the extension, allow it to install **ZLS** (the Zig language server) automatically. Doing this step enables autocomplete and inline errors.

### 5. Download the datasets
See [Data Setup](#data-setup) above.

### 6. Verify everything works
```bash
python src/main.py
```
Should print `7`.

## Development

### Rebuilding after Zig changes
Only need to run the following rebuild command line after modifying `.zig` files:
```bash
python -m ziglang build
```
Python files do **not** require a rebuild.

### Calling Zig functions from Python
All ctypes declarations are in `declarations.py`. Import from there in your scripts:
```python
from declarations import lib

result = lib.some_function(arg1, arg2)
```

### Adding a new Zig function
1. Write and `export` the function in `src/main.zig`
2. Rebuild: `python -m ziglang build`
3. Add any missing `argtypes` and `restype` declarations to `declarations.py`
